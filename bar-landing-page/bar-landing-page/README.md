# Bar Landing Page

A single-page landing site for a fictional neighborhood bar, built to practice **visual design, layout, and scroll-based interactions**. The aesthetic is a "whiskey noir" look — dark backgrounds, warm accent tones, editorial typography, and subtle motion.

## Live Demo

Enable GitHub Pages (Settings → Pages → Source: `main`) and the page runs at `https://ralph192.github.io/bar-landing-page/`.

## Features

- **Cinematic hero** section with layered backgrounds and animated elements
- **Story / about** section with editorial typography
- **Events grid** highlighting recurring nights and activities
- **Drinks menu** with a featured-pricing layout
- **Testimonials** as a card layout
- **Staff spotlight** cards
- **Hours, location & contact** section
- **Scroll-triggered reveal animations** via `IntersectionObserver`
- **Sticky navigation** that shrinks on scroll
- Fully responsive

## Tech Stack

- **HTML5** — single-page semantic structure
- **CSS3** — custom properties, gradients, transitions, responsive layout
- **JavaScript (vanilla)** — scroll animations and sticky-nav behavior
- **Google Fonts** — Playfair Display, DM Mono, Libre Baskerville

## Running Locally

Open `index.html` in any modern browser.

## Notes

This is a design concept for a **fictional** business. All names, contact details, reviews, and other content are placeholder/invented for demonstration purposes.
